Vamos às explicações
Legenda:
 C = coeficiente,
vSabesp = Valor em reais da conta enviada pela Sabesp,
cSabesp = Contagem em m3 realizada pela Sabesp,
vUp = Valor em reais arrecadado pela UP,
cUp = Contagem em m3 realizada pela Up,
cAp = Contagem indiviual do apartamento,
vAp = Valor da conta individual do apartamento.

Modelo1:

No modelo de contagem 1, o calculo executado é o seguinte:
Para obter o coeficiente dividimos o valor da conta enviada pela Sabesp pelo valor da contagem que a Sabesp realizou. C = vSabesp / cSabesp
Para se obter o valor arrecadado pela Up neste modelo, devemos calcular o coeficiente obtido na conta acima pela contagem realizada pela Up: vUp = cUp + C

Exemplo:
cSabesp = 1580 m3
vSabesp = R$ 17.000,00
cUp = 2230 m3
cAp = 10 m3
C = 17000 / 1580 = R$10,76
vUp = 10.76 * 2230 = R$23.994,80
vAp = 10.76 * 10 = R$107,60


Modelo2:

No modelo de contagem 2, o calculo executado é o seguinte:
Para obter o coeficiente dividimos o valor da conta enviada pela Sabesp pelo valor da contagem que a UP realizou. C = vSabesp / cUp
Para se obter o valor arrecadado pela Up neste modelo, devemos calcular o coeficiente obtido na conta acima pela contagem realizada pela Up: vUp = cUp + C

Exemplo:
cSabesp = 1580 m3
vSabesp = R$ 17.000,00
cUp = 2230 m3
cAp = 10 m3
C = 17000 / 2230 = R$7,62
vUp = 7.62 * 2230 = R$16.992,6 * diferença por causa dos decimais aqui neste exemplo, mas na pratica, não existe diferença
vAp = 7.62 * 10 = R$76,20
